Sobre Tec-IA

Conoce más sobre nuestro asistente inteligente

El chatbot se basa en el procesamiento del Lenguaje Natural (PLN) para comprender, interpretar y extraer información clave de tus documentos. Esto nos permite ofrecerte resúmenes, identificar entidades importantes y responder a tus preguntas de manera contextual.

Al chatear con un PDF, Tec-IA procesa el documento para crear una base de conocimiento. Luego, cuando haces una pregunta, el sistema busca la información más relevante dentro del texto para darte una respuesta coherente y precisa, citando las fuentes si es necesario.

Desde estudiantes que necesitan resumir textos académicos y extraer datos de informes, Tec-IA es una herramienta versátil.
Utiliza el modelo facebook/bart-large-cnn para realizar los resúmenes, para extraer las palabras claves o keywords se usa el modelo de Keybert

1. Sube un archivo o pega un texto.
2. Haz preguntas directas sobre el contenido.
3. Utiliza las herramientas de análisis para obtener resúmenes o identificar datos clave.
4. Explora las diferentes funciones del menú para descubrir todo lo que Tec-IA puede hacer por ti.

Somos un proyecto de código abierto y nos encantaría contar con tu ayuda. Puedes encontrar nuestro repositorio en GitHub, donde puedes reportar errores, sugerir nuevas funcionalidades o incluso contribuir con tu propio código.
